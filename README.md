# Zazi-SAT-Prep

SAT prep app designed to help students reach 1550+ by providing targeted practice,
step-by-step explanations, and realistic practice tests.

## Features
- Targeted SAT practice questions
- Step-by-step solutions
- Weakness tracking
- Realistic timed tests

## Tech Stack
- React
- JavaScript
- Node.js

## Installation
1. Clone the repo
2. Run npm install
3. Run npm start
